# Import an SPDR fund

Wrapper around
[`load_fund()`](https://stantraykov.github.io/fundsr/reference/load_fund.md)
for SPDR files.

## Usage

``` r
spdr(ticker, file = NULL, benchmark = NULL)
```

## Arguments

- ticker:

  Fund ticker.

- file:

  Optional filename override.

- benchmark:

  Optional benchmark key.
