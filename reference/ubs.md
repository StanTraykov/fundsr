# Import a UBS fund

Wrapper around
[`load_fund()`](https://stantraykov.github.io/fundsr/reference/load_fund.md)
for UBS files.

## Usage

``` r
ubs(ticker, file = NULL, benchmark = NULL)
```

## Arguments

- ticker:

  Fund ticker.

- file:

  Optional filename override.

- benchmark:

  Optional benchmark key.
